# 查看代码覆盖率

  • GCC + gcov (cmake 一般带了) + lcov
  1. 修改 CMakeLists.txt
  1. 构建并运行测试
    …
    cmake … -DENABLE_COVERAGE=ON
    …
  2. 生成可视化覆盖率报告（HTML）

或者直接

然后再 build 目录下的 coverage_html 目录下查看 index.html，可以点开具体文件查看哪些行没有覆盖到

# 如何提高单元测试的覆盖率

分析当前覆盖率报告，找出未执行的代码路径，然后编写新的测试用例来触发这些路径。

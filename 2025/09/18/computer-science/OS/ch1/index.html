# Ch1

计算机系统自下而上可以大致分为硬件、操作系统、应用程序、用户
OS 管理各种硬件，为应用程序提供基础，充当计算机硬件与用户之间的中介，是计算机系统中最基本的系统软件

# 基本概念

# 操作系统的功能

  1. 管理计算机系统资源
    处理机管理（进程 / 线程管理）、存储器管理、设备管理、文件管理，向用户提供接口
  2. 作为用户与硬件系统之间的接口
  • 命令接口（联机控制方式和脱机控制方式）
    • 联机命令接口也称为交互式命令接口，由一组键盘操作命令组成
    • 脱机命令接口也称批处理命令接口，由一组作业控制命令组成
  • 程序接口
    • 由一组系统调用组成
  1. 实现了对计算机资源的补充
    虚拟机

# 操作系统的特征

最基本的两个：并发与共享

  1. 并发
    多道程序交替运行（和并行不一样）
  2. 共享
  • 互斥共享
  • 同时访问
  1. 虚拟
  2. 异步

# 手工操作阶段（无操作系统）

# 批处理阶段

  1. 单道批处理
    单道程序
  • 自动型：一批作业自动地逐个运行
  • 顺序性：顺序进入内存、顺序运行
  • 单道性：内存中仅有一道程序运行，一道程序运行完才换入下一道
  1. 多道批处理
    多道，宏观上并行，微观上串行
  • 多道：内存中同时存放多道相互独立的程序
  • 宏观上并行
  • 微观上串行：轮流占有 CPU，交替执行
    缺点：用户响应时间长，不提供人机交互能力，用户既不能了解自己的程序的运行情况，又不能控制计算机
